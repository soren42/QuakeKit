QuakeKit / Brand Standards / v1.0

Instrument, not webpage.

QuakeKit turns the DK ultrawide touch panel into a themeable command surface. This companion defines the implementation rules; the supplied PDF remains the visual and print authority.

Open Print Edition (PDF)
01 / Voice

Terse, technical, neutral

Terse

Dense and scannable. Prefer 48 min left to explanatory prose.

Technical

Use real units and state names: connected, pong 0.4s, exit 1.

Neutral

Labels are nouns; actions are verbs. Use sentence case in UI and uppercase only for kickers.

State the fact: Plugin action failed - exit 1. Avoid marketing language, cute system copy, emoji, and vague errors.

05 / Theme and motion contract

Nine themes, one behavioral system

Cyberpunk NeonDefault high contrast
Focus DarkRestrained operational
Weather GlassAmbient passive
Vantage Dark / LightProfessional technical grid
Aqua Glass Dark / LightmacOS vibrancy
BeOS Light / DarkBeveled retro

Theme changes remap token values and surface treatment, not interaction geometry. The menu widget owns all on-panel chrome; the shared backpage remains stable beneath it.

Motion: focus snaps and content may cross-fade around 160ms. Neon pulses, Aqua/Weather fade, Vantage/BeOS/Focus remain static. Honor reduced motion and never animate decoratively.
06 / Panel rules

Designed for 1920 x 480, knob and touch

The DK panel is landscape and dark. Maintain a stable top-level menu shell, a clear navigation rail/tab/dock, and a shared content region. Focus follows a discrete knob ring; touch activates the same declared actions. Preserve contrast and avoid controls that depend on a mouse cursor.

Host: display/lifecycle/HID/data · Menu: chrome/navigation/Home · Shared library: standard pages/applet frame · Plugin: applet body
