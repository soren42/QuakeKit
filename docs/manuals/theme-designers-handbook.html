1. What Themes Own

Themes own presentation, not behavior. A theme should describe how QuakeKit looks and how semantic states map to visual treatment. Functional plugins still own data, actions, and integration logic.

Themes OwnPlugins Own
Color tokens, semantic colors, typography, component styles, background assets, layout preferences, knob-ring visual mapping.Data sources, actions, settings, view declarations, external services, semantic status requests.

2. Package Layout

cyberpunk-neon.quakekittheme/
  theme.json
  assets/
    neon-circuit.svg

Only theme.json is required. Asset files are optional but should be listed in the assets array when included.

3. Palette And Semantics

Define token colors first, then map semantic host roles to those tokens. This keeps native UI, web applets, and future hosts aligned.

Semantic RoleRequiredUse
backgroundYesMain panel background.
surfaceYesBase cards, rows, and controls.
surfaceRaisedYesSelected or elevated UI surfaces.
borderYesTile, tab, chart, and row boundaries.
textPrimaryYesMain labels and values.
textSecondaryYesSecondary labels and captions.
accentYesFocus, selection, and key actions.
success / warning / dangerYesHealth/status states.
 
"palette": {
  "mode": "dark",
  "colors": {
    "void": { "value": "#04070D", "role": "Panel background" },
    "cyan": { "value": "#39F5FF", "role": "Primary neon accent", "configurable": true },
    "magenta": { "value": "#FF5CDB", "role": "Secondary accent", "configurable": true }
  },
  "semanticColors": {
    "background": "void",
    "surface": "#080D18",
    "surfaceRaised": "#11182A",
    "border": "cyan",
    "textPrimary": "#F4F9FF",
    "textSecondary": "#AAB7D4",
    "accent": "cyan",
    "success": "#A7FF57",
    "warning": "#FFE66D",
    "danger": "#FF6B5C"
  }
}

4. Typography And Metrics

Keep panel typography dense and legible. The DK panel is wide and short, so overly large type wastes vertical space quickly.

"typography": {
  "displayFont": "Avenir Next Condensed",
  "bodyFont": "SF Pro Text",
  "monoFont": "Menlo",
  "scale": {
    "caption": 10,
    "body": 13,
    "title": 18,
    "value": 28
  }
},
"metrics": {
  "cornerRadius": 7,
  "borderWidth": 1,
  "spacing": 8,
  "density": "compact"
}

5. Components

Component styles are token overrides for common host surfaces. They should reference palette tokens or semantic colors, not one-off values scattered across the file.

ComponentTypical Use
tileWidget tiles and app action panels.
tabTop page labels and segmented navigation.
statusRowRuntime rows and compact settings rows.
gaugeCPU, memory, weather, or system bars.
chartLine charts, sparklines, and telemetry graphs.

6. Layout Hints

Themes can express preferred layout style without forcing every plugin to draw an entire page.

FieldValuesUse
defaultPageStylegrid, fullScreen, halfAndGrid, twoHalves, thirds, quartersDefault composition.
widgetGridcolumns, rowsPreferred widget grid.
appletGridcolumns, rowsPreferred applet grid.
splitRatio0.2 to 0.8Split-screen proportion.

7. Image And Design Assets

Themes may include images, fonts, sounds, CSS, or scripts. Native host components should not execute theme scripts; script assets are for web/plugin surfaces once the bridge is stable.

"assets": [
  {
    "id": "circuit",
    "kind": "image",
    "path": "assets/neon-circuit.svg",
    "role": "background",
    "fit": "cover",
    "opacity": 0.28
  }
]

Prefer SVG or high-resolution PNG assets that tolerate downscaling to the DK panel. Avoid low-contrast dark artwork behind dense telemetry.

8. Knob Ring Presentation

The knob LED ring belongs to the theme as a presentation map. Plugins request semantic states; the runtime arbitrates priority and TTL; the theme maps the winning state to color, intensity, and animation.

"hardware": {
  "knobRing": {
    "enabled": true,
    "idle": { "color": "cyan", "intensity": 0.18, "animation": "solid" },
    "focus": { "color": "cyan", "intensity": 0.75, "animation": "pulse" },
    "success": { "color": "#A7FF57", "intensity": 0.7, "animation": "solid" },
    "warning": { "color": "#FFE66D", "intensity": 0.85, "animation": "pulse" },
    "danger": { "color": "#FF6B5C", "intensity": 1.0, "animation": "flash" },
    "progress": { "color": "magenta", "intensity": 0.8, "animation": "progress" }
  }
}

9. Configurable Options

Options expose safe visual customization controls in the settings UI. Keep options targeted and understandable.

"options": [
  {
    "id": "accent",
    "title": "Accent Color",
    "type": "color",
    "target": "palette.colors.cyan.value",
    "defaultValue": "#39F5FF"
  },
  {
    "id": "density",
    "title": "Density",
    "type": "choice",
    "target": "metrics.density",
    "defaultValue": "compact",
    "choices": ["compact", "standard", "comfortable"]
  }
]

10. Complete Theme Example

{
  "$schema": "../../schemas/theme-plugin.schema.json",
  "id": "example_neon",
  "name": "Example Neon",
  "version": "1.0.0",
  "apiVersion": "0.1",
  "kind": "theme",
  "palette": {
    "mode": "dark",
    "colors": {
      "base": { "value": "#05070D" },
      "surface": { "value": "#0B1020" },
      "cyan": { "value": "#39F5FF", "configurable": true },
      "pink": { "value": "#FF5CDB", "configurable": true }
    },
    "semanticColors": {
      "background": "base",
      "surface": "surface",
      "surfaceRaised": "#11182A",
      "border": "cyan",
      "textPrimary": "#FFFFFF",
      "textSecondary": "#AAB7D4",
      "accent": "cyan",
      "success": "#A7FF57",
      "warning": "#FFE66D",
      "danger": "#FF6B5C"
    }
  },
  "metrics": { "cornerRadius": 7, "borderWidth": 1, "spacing": 8, "density": "compact" },
  "layout": { "defaultPageStyle": "fullScreen", "widgetGrid": { "columns": 4, "rows": 2 } },
  "options": [
    { "id": "accent", "title": "Accent Color", "type": "color", "target": "palette.colors.cyan.value", "defaultValue": "#39F5FF" }
  ]
}

11. Validate And Install

swift run quake-probe --install-package ./ExampleNeon.quakekittheme
swift run quake-probe --install-package ./ExampleNeon.quakekittheme.tar.gz

12. Current Gaps

  • Theme asset existence is structurally declared but not fully enforced by validation.
  • CSS/script theme assets need a stricter bridge contract before third-party execution is recommended.
  • The settings UI exposes first-pass controls; richer color pickers and preview tooling are still needed.
  • Printed visual examples should be refreshed with final v1.0 screenshots before publication.
